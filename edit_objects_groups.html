Список групп (заполнение полей таблицы objects_groups) в виде иерархического дерева с возможностью добавления, редактирования и удаления, при клике на кнопку редактирования попадаем в отдельную форму редактирования группы.
При добавлении группы создается папка группы с именем кодового обозначения группы внутри папки родительской группы. Если родителя нет, то папка создается в папке /actions/addObject, путь до которой указывается в качестве константы в начале скрипта панели администрирования. При удалении группы папка группы удаляется (удалить группу можно, только если к ней не привязан ни один объект)

  • Добавление группы
  • Редактирование существующей группы
  • Удаление существующей группы
